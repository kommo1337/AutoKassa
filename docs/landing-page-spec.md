# ТЗ: Landing Page для AutoKassa

> Цель: создать современный одностраничный сайт на GitHub Pages, который представляет десктопное приложение AutoKassa и побуждает к скачиванию.

---

## 1. Общая концепция

- **Стиль:** светлый, минималистичный, с отсылками к UI самого приложения
- **Настроение:** профессиональный, доверительный, современный
- **Язык:** русский
- **Хостинг:** GitHub Pages (ветка `master`, папка `/docs`)
- **URL:** `https://kommo1337.github.io/AutoKassa`

## 5. Интерактивность и анимации

- **Появление секций:** при скролле — плавное выезжание снизу (`opacity` + `translateY`), библиотека: `AOS` или нативный `IntersectionObserver`
- **Кнопки:** при наведении — смещение вверх на `–2px` + усиление тени
- **Скриншоты:** при наведении — лёгкое увеличение `scale(1.02)` + увеличение тени
- **Навигация:** плавный скролл к якорям (`scroll-behavior: smooth`)
- **Без тяжёлых эффектов:** никакого параллакса или WebGL — страница должна летать даже на слабом железе

---

## 6. Адаптивность (Responsive)

| Разрешение | Поведение |
|-----------|-----------|
| ≥ 1024 px | Полный макет: Hero с текстом слева и скриншотом справа, 3 колонки скриншотов и карточек |
| 768 – 1023 px | Hero в колонку (текст сверху, скриншот снизу), 2 колонки карточек |
| < 768 px | Одна колонка, мобильное меню-гамбургер (или скрытая навигация), компактные отступы |

## 7. Техническая реализация

- **HTML5 + CSS3** (без фреймворков, чистый код для скорости загрузки)
- **Google Fonts:** `Lato` (weights: 400, 700)
- **Иконки:** emoji или `lucide-static` SVG
- **Анимации:** нативный CSS + `IntersectionObserver` (без jQuery)
- **Скриншоты:** переиспользовать `screenshots/dashboard.png`, `operations.png`, `reports.png`
- **GitHub Pages:** страница собирается из папки `docs/` ветки `master`
